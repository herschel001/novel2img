# Novel2Img 项目结构

## 📂 最终项目结构

```
novel2img/
├── README.md                    # 项目说明文档
├── 需求文档.md                  # 产品需求文档  
├── 测试指南.md                  # 安装和测试指南
├── manifest.json               # Chrome插件配置文件
├── background/                 # 后台脚本目录
│   └── background.js          # 后台服务逻辑
├── popup/                     # 弹窗界面目录
│   ├── popup.html            # 弹窗HTML结构
│   ├── popup.css             # 弹窗样式文件
│   └── popup.js              # 弹窗交互逻辑
├── lib/                       # 公共库目录
│   ├── api-client.js         # API调用封装
│   └── utils.js              # 工具函数库
└── icons/                     # 图标资源目录
    ├── icon.svg              # SVG图标源文件
    ├── icon16.png            # 16x16 图标
    ├── icon48.png            # 48x48 图标
    └── icon128.png           # 128x128 图标
```
